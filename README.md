# Fraud Detection for E-commerce and Bank Transactions

## Overview
This project focuses on detecting fraudulent transactions in:
- E-commerce transactions
- Bank credit card transactions

The solution addresses class imbalance, feature engineering, geolocation analysis,
and model explainability using SHAP.
